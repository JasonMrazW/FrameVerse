欢迎来到我的技术博客！这是用 [Hugo](https://gohugo.io/) 搭建的第一篇文章。

## 为什么选择 Hugo

- **构建极快**：基于 Go，秒级生成成百上千篇文章。
- **部署简单**：输出纯静态文件，可托管到任意平台。
- **Markdown 写作**：专注内容，无需操心样式。

## 代码示例

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Blog!")
}
```
